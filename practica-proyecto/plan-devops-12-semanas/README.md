# Plan DevOps 12 Semanas

Este directorio organiza el estudio por fases y semanas.
Cada semana tiene su `README.md` con:
- objetivo
- tareas
- entregable
- checklist

## Orden de uso

1. Leer `00-guia/README.md`
2. Ejecutar semana actual
3. Registrar evidencia en `99-tracking/bitacora-semanal.md`
4. Cerrar semana en `99-tracking/progreso.md`
