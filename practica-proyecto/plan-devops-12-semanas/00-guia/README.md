# Guia de Trabajo

## Regla principal
70% practica, 30% teoria.

## Rutina semanal
- Lunes: teoria guiada
- Martes: practica
- Miercoles: practica
- Jueves: integracion
- Viernes: evidencia y retro

## Evidencia minima por semana
- 1 resumen tecnico
- 1 practica ejecutada
- 1 commit o log de comandos

## Stack recomendado para arrancar
- AWS
- Docker
- GitHub Actions
- Kubernetes
- Terraform
- Prometheus + Grafana
